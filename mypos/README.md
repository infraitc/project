## Install Angular CLI
```
npm cache clean --force
npm install -g @angular/cli@latest
```

## Create Project
```
ng n mypos
```

## Start Project
Default Port 4200
```
ng s -o
ng s -o | --port
npm start
```

## Short Key
Go to file
```
command + p (mac)
ctrl + p (windows)
```

Switch between .ts and .html (angular2-switcher extension)
```
alt + shift + o (mac)
alt + o (windows)
```

Go to symbol
```
command + shift + o (mac)
ctrl + shift + o (windows)
```
